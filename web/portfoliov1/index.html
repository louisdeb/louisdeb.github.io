OpenGL 3D Vector Visualiser

01/11/2013 - 25/04/2014

View raw on GitHub

View project report

Using OpenGL, Tao.freeglut, and written in C#, this project was commissioned by an A2 Further Maths teacher to aid teaching 3D vectors in the classroom.

The project was extensively covered in the report, including design and appraisal.

To further aid visualisation of the vectors, translation, rotation and 3D anaglyph (use with red-cyan glasses) features were included.
